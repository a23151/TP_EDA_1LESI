\doxysubsection{Detailed Description}
Funções desenvolvidas para a fase 1 do projeto, para gestão de operações e máquinas de um job. 

Funcoes desenvolvidas para gestao de jobs. Os jobs estão organizados numa estrutura de ABP.

Funcoes desenvolvidas para uma proposta de escalonamento.
